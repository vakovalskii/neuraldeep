# find-skills

Навык для AI-агентов: поиск и установка навыков из каталога skillsbd.

## Когда использовать

Активируй этот навык, когда пользователь:
- Просит найти навык или плагин для конкретной задачи
- Хочет расширить возможности агента (например, "мне нужен навык для работы с Kubernetes")
- Спрашивает "какие навыки есть для X?"
- Просит установить или подключить навык по имени
- Хочет узнать, какие навыки уже установлены

## Как искать навыки

### 1. Через CLI (рекомендуемый способ)

```bash
# Поиск по ключевому слову
npx skillsbd search "docker"

# Поиск по категории
npx skillsbd search --category devops

# Поиск по автору
npx skillsbd search --author anthropic
```

### 2. Через API

```bash
# GET-запрос к каталогу
curl "https://skillsbd.ru/api/skills?q=docker"
```

Ответ возвращает JSON-массив навыков с полями: `name`, `description`, `author`, `installs`, `featured`, `categories`, `repo`.

### 3. Через веб-каталог

Направь пользователя на https://skillsbd.ru для визуального поиска и фильтрации навыков.

## Установка и управление навыками

```bash
# Установить навык (короткий формат)
npx skillsbd add owner/repo/skill-name

# Установить навык (полный формат с URL)
npx skillsbd add https://github.com/owner/repo --skill skill-name

# Посмотреть установленные навыки
npx skillsbd list

# Удалить навык
npx skillsbd remove skill-name
```

## Как проверить качество навыка

Перед тем как рекомендовать навык пользователю, оцени его по следующим критериям:

1. **Количество установок** (`installs`) -- навыки с > 100 установками, как правило, стабильны
2. **Автор** (`author`) -- известные авторы и организации заслуживают больше доверия
3. **Значок "Рекомендовано"** (`featured: true`) -- навык прошёл проверку и одобрен командой skillsbd
4. **Дата обновления** -- навыки, обновлённые менее 3 месяцев назад, предпочтительнее
5. **Наличие описания и документации** -- качественный навык всегда имеет SKILL.md

Приоритет: featured > количество установок > дата обновления > автор.

## Как представить результаты пользователю

Когда нашёл подходящие навыки, выведи их в таком формате:

```
Найдено N навыков по запросу "query":

1. **owner/skill-name** -- Краткое описание навыка
   Установок: 500 | Автор: owner | Рекомендовано
   → npx skillsbd add owner/repo/skill-name

2. **owner2/skill-name2** -- Краткое описание
   Установок: 120 | Автор: owner2
   → npx skillsbd add owner2/repo2/skill-name2
```

Правила:
- Показывай не более 5 результатов за раз
- Сортируй по релевантности, затем по количеству установок
- Если навык featured -- укажи это явно
- Всегда предоставляй готовую команду для установки

## Распространённые категории

| Категория | Примеры запросов |
|-----------|-----------------|
| `devops` | Docker, Kubernetes, CI/CD, Terraform |
| `web` | React, Next.js, API, REST, GraphQL |
| `data` | PostgreSQL, MongoDB, Redis, аналитика |
| `testing` | тесты, QA, pytest, Jest |
| `security` | аудит, уязвимости, SAST |
| `docs` | документация, README, changelog |
| `ai` | ML, LLM, промпты, fine-tuning |
| `mobile` | iOS, Android, React Native, Flutter |
| `infra` | AWS, GCP, Azure, облако |
| `productivity` | git, автоматизация, рефакторинг |

## Когда ничего не найдено

Если поиск не дал результатов:

1. Попробуй переформулировать запрос (синонимы, английские термины)
2. Используй более общую категорию
3. Проверь написание
4. Предложи пользователю:
   - Поискать вручную на https://skillsbd.ru
   - Создать собственный навык и опубликовать его в каталоге
   - Описать задачу подробнее, чтобы подобрать альтернативу

```
По запросу "X" навыки не найдены.
Попробуйте:
- Изменить запрос (например, "Y" вместо "X")
- Посмотреть каталог: https://skillsbd.ru
- Создать свой навык: https://skillsbd.ru/docs/create-skill
```
